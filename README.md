
# Frogger - Классическая Аркадная Игра

Классическая аркадная игра Frogger, воссозданная на современном JavaScript с использованием модульной архитектуры и принципов ООП.

## Быстрый Запуск
1. Скачайте все файлы проекта в одну папку
2. Откройте `index.html` в современном браузере (Chrome, Firefox, Edge)
3. Введите имя игрока и нажмите "START GAME"
4. Начните играть сразу - не требуется сборка или дополнительные зависимости

## Управление
- Стрелки или WASD - движение лягушки вверх/вниз/влево/вправо
- P - пауза/продолжение игры
- Свайпы на мобильных устройствах - альтернативное управление

## Геймплей
Цель игры - безопасно провести лягушку через оживленную дорогу и реку к домам на противоположной стороне.

### Основные Правила:
- Машины убивают - избегайте столкновений с движущимся транспортом
- Река опасна - лягушка должна прыгать по бревнам, иначе утонет
- Двигайтесь по клеткам - все перемещения дискретны
- Заполняйте дома - цель каждого уровня - занять все 5 домов
- Бревна двигают - стоя на бревне, лягушка движется вместе с ним

### Уровни Сложности:
- Уровень 1 - умеренное движение транспорта и бревен
- Уровень 2 - увеличенная скорость и количество препятствий

### Система Очков:
- Достижение дома: +100 очков × уровень
- Временный дом (верхняя граница): +50 очков
- Завершение уровня: +500 очков

## Технические Особенности
- Чистый JavaScript с нативными ES-модулями
- Объектно-ориентированный подход с четким разделением ответственности
- Модульная архитектура для легкого расширения и поддержки
- Адаптивный дизайн с поддержкой мобильных устройств
- Дискретное движение - все объекты перемещаются по клеточной сетке
- Реалистичная физика - лягушка движется вместе с бревнами

## Механики Движения
- Лягушка перемещается ровно на одну клетку за ход
- Машины движутся с разной скоростью в обоих направлениях
- Бревна перемещаются по реке, увлекая за собой лягушку
- Коллизии рассчитываются по центру объектов для точности

## Стратегия Успеха
1. Планируйте маршрут - оцените движение машин перед выходом на дорогу
2. Используйте бревна - правильно рассчитывайте прыжки на движущиеся платформы
3. Следите за временем - препятствия движутся непрерывно
4. Приоритет безопасности - иногда лучше подождать, чем рисковать
5. Заполняйте дома равномерно - не оставляйте сложные позиции на потом
